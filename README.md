# SkillBoost 

**Платформа для подготовки к техническим собеседованиям:** роудмап, тренажёр собеседований, база вопросов и live‑coding.

[![Веб-сайт](https://skillboost-bt1n.vercel.app/)

## О проекте

SkillBoost — это единое пространство для разработчиков, которые хотят системно подготовиться к техническим интервью. Здесь вы найдёте всё необходимое, чтобы уверенно проходить собеседования в IT-компании.

## Возможности

- **Роудмап** — пошаговый план подготовки под разные грейды и технологии
- **Тренажёр собеседований** — симуляция реального интервью
- **База вопросов** — сотни реальных вопросов с ответами и разборами
- **Live‑coding** — интерактивная среда для решения алгоритмических задач прямо в браузере
